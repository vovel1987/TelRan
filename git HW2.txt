Используя ветвление, Вы отклоняетесь от основной линии разработки и продолжаете работу независимо от неё, не вмешиваясь в основную линию.

В процессе разработки мы можем использовать главную ветку и второстепенную для каких то изменений.После проверки изменений мы можем соединить две ветки.
Команды переименования веток:
-git branch --move  
Команды создания и переключения на новую ветку:
-git branch
-git checkout -b

Удаление веток:
-git branch -d 
-git branch -D

Слияние веток:
git merge
